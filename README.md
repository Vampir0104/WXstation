Required before using this weather station:

Python 3.11
Streamlit
Request


To use, create an enviroment for Python 3.11, use CD to point to the directory this file is in, then type in 'Streamlit run WXstation.py'

Use IATA codes to call weather reports and TAFs/forecasts.  IE: KSLC, KDTW, KORD
Reports are returned in raw form.
